# WPF Library Management System

## 📌 Overview
Ứng dụng quản lý thư viện xây dựng bằng WPF theo mô hình MVVM.

## 🛠 Technologies
- C#
- WPF
- MVVM Pattern
- SQL Server (LocalDB)

## 🚀 Features
- Thêm / Sửa / Xóa sách
- Tìm kiếm theo mã hoặc tên
- Validation dữ liệu
- Thống kê tổng số sách

## 🏗 Architecture
- Model
- View
- ViewModel

## ▶ How to Run
1. Clone repository
2. Mở file .sln bằng Visual Studio
3. Restore database
4. Run project
